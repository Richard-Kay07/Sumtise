# Public Assets Directory

## Logo Image

Place your Sumtise logo image file here with the filename: **`logo.png`**

### Supported Formats
- PNG (recommended)
- SVG
- JPG/JPEG
- WEBP

### File Requirements
- **Filename**: `logo.png` (or update the references in `src/components/logo.tsx` and `preview.html` if using a different name)
- **Location**: `/public/logo.png`
- **Recommended size**: 512x512px or higher (for crisp display at all sizes)
- **Background**: Transparent PNG recommended for best results

### Usage
The logo will be automatically used in:
- Header navigation
- Sign-in page
- Sign-up page
- All pages using the `<Logo />` component

### Updating the Logo
1. Place your logo file as `logo.png` in this directory
2. The logo will be automatically available at `/logo.png`
3. No code changes needed (unless using a different filename)
